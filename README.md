# wangsta

- Distributed, cross DB, Counter (ID).

**Easy to play, hard to master.**

## Requirements

*Goal is simple.*

- Consistent, cross DB Engines, reliable Counter (ID), which can be used for poly-db interop.

*One ID which joins them all.*
