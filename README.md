# wangsta

- Cross Database (Wanna Be) Lock Free Counter (ID).

**Easy to play, hard to master.**

## Requirements

*Goal is simple.*

- Consistent, cross DB Engines, reliable ID, which can be used for polyglot-db inter-op.

*One ID which joins them all.*

## Integrations

- PostgreSQL
- Couchbase
